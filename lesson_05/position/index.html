Position

  • static - это значение по умолчанию
  • relative - распологается относительно нормального положения, как если бы элемент был статическим. Другие элементы ведут себя, как если бы релатив оставался на своем месте, то есть они схлопываются
  • absolute - мы рапсполагаем элемент относительно его ближайшего позиционированного родителя(position не статик) - сам элемент исключается из естественного потока (связка с релатив или другим позиционированным)
  • fixed - когда элемент позиционируется относительно view-port - окна видимости - используем два из: top right left bottom
  • sticky - липкий - сначала ведет себя как статик, а после как absolute (nb: не забыть указать top, иначе не будет залипать)
